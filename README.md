# YinYang-SWE-MacCormack42-3rd-Stabilization_Strategy
Fourth-order compact MacCormack solver for the shallow water equations on a Yin–Yang overset grid with fourth-order Runge–Kutta time integration. The code implements Strategy 3, a stabilization technique based on enhanced Yin–Yang overlap and Yin-centered derivative evaluation, and is validated against two benchmark test cases.
